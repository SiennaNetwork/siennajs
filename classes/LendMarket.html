simulateLiquidation

  • simulateLiquidation(borrower: string, collateral: string, amount: string, block?: number): Promise<LendSimulatedLiquidation>

  • Dry run a liquidation returning a result indicating the amount of collateral which would be seized, and whether the liquidation would be successful depending on whether the borrower posseses the seized collateral amount. If it wouldn't, throw any other error that might occur during the actual liquidation.

    If you haven't taken the close factor into account already, you might want to look for an error that starts with "Repay amount is too high." as that indicates that you are trying to liquidate a bigger portion of the borrower's collateral than permitted by the close factor.

    Parameters

    • borrower: string

      Param

      the ID corresponding to the borrower to liquidate.

    • collateral: string

      Param

      the collateral market address to receive a premium on.

    • amount: string

      Param

      the amount to liquidate by.

    • Optional block: number

    Returns Promise<LendSimulatedLiquidation>
